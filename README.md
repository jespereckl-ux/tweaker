# 🎮 Safe Python-Based FPS Gaming Tweaker

A professional Windows FPS optimization application designed for competitive gaming. **Safe, transparent, and completely reversible.**

## Philosophy

This tool prioritizes:
- **Safety**: No security bypasses, antivirus stays enabled
- **Transparency**: Every change is logged and explained
- **Performance**: Measurable FPS improvements for competitive games
- **Reversibility**: Full rollback/restore capabilities
- **User Control**: No hidden modifications

## Features

✅ System hardware detection and profiling
✅ Safe, reversible optimizations with risk ratings
✅ Comprehensive rollback/restore system
✅ Detailed logging and audit trails
✅ Modern desktop UI with dashboard
✅ Smart recommendation engine
✅ Before/after benchmarking
✅ Profile management (save/load configurations)

🛡️ **Never disabled**:

## Quick Start

```bash
# Clone the repository
git clone https://github.com/jespereckl-ux/tweaker.git
cd tweaker

# Install dependencies
pip install -r requirements.txt

# Run the application (requires admin)
python main.py
```

## Risk Levels

- **SAFE**: No known issues, no reboot required
- **LOW RISK**: Minor behavior changes, generally safe
- **MEDIUM RISK**: May affect background apps, reversible
- **HIGH RISK**: May require reboot or affect compatibility
